# Personel-Takip-Otomasyonu
Bu proje, kurum veya işletmelerde çalışan personellerin giriş-çıkış saatleri, izin durumları ve görev bilgilerini takip etmeyi amaçlayan bir Personel Takip Otomasyonu sistemidir. Proje, kullanıcı dostu bir arayüz ve veritabanı entegrasyonu ile personel yönetimini kolaylaştırır.
